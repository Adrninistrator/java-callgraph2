# 限制表达式忽略的数据输出行数
通过配置文件 _javacg2_config/config.properties 中的配置参数 el.ignore.data.max.line.num 可以限制表达式忽略的数据输出行数

# 表达式调试方式
将配置文件 _javacg2_config/config.properties 的 el.debug.mode 参数设置为 true 可以使表达式执行时开启调试模式，会在应用日志中输出表达式执行时的详细信息
